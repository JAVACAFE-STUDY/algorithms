# 13주차 알고리즘 문제 풀이

## 문제 목록

### 1. 완주하지 못한 선수

-   **파일**: `42576.ts`
-   **테스트**: `42576.test.ts`
-   **링크**: https://school.programmers.co.kr/learn/courses/30/lessons/42576
-   **요약**: 마라톤 참가자 배열과 완주자 배열이 주어질 때, 완주하지 못한 선수 1명을 찾는 문제
-   **접근**: 해시맵(Map)으로 참가자 이름별 카운트를 저장한 후, 완주자 배열을 순회하며 카운트를 차감. 카운트가 0보다 큰 선수가 완주하지 못한 선수
-   **시간복잡도**: O(n) – participant 순회 O(n) + completion 순회 O(n) + 맵 순회 O(n)
-   **공간복잡도**: O(n) – 참가자 수만큼 맵에 저장

## 학습 포인트

### 해시 테이블 (완주하지 못한 선수)

-   **카운팅 패턴**: 동명이인이 있을 수 있으므로 단순 존재 여부가 아닌 출현 횟수를 카운트
-   **차집합 개념**: 참가자 집합에서 완주자 집합을 빼는 개념을 해시맵 카운트로 구현
-   **시간복잡도**: 정렬 O(n log n) 대신 해시맵 O(n)으로 효율적 해결
